# Login Slider Authentication UI

## Project Overview

This project is a basic authentication interface that features a **sliding transition between Login and Sign Up forms**. The interface allows users to easily switch between the two forms through a smooth slider animation.

The purpose of this project was to practice creating interactive and responsive user interfaces while improving front-end development skills.

---

## Technologies Used

* TypeScript
* HTML
* CSS

---

## Features

* Sliding animation between Login and Sign Up forms
* Responsive layout
* Interactive input fields
* Password visibility toggle
* Clean and modern UI design

---

## How to Run the Project

### Step 1: Download or Clone the Project

Download the project files or clone the repository to your computer.

### Step 2: Open the Project Folder

Open the project folder using **Visual Studio Code** or another code editor.

### Step 3: Compile TypeScript

If the project includes TypeScript files, compile them by running the following command in the terminal:

tsc

This will convert the TypeScript files into JavaScript so the browser can run them.

### Step 4: Open the Project

Locate the **index.html** file in the project folder.

### Step 5: Run the Website

Double-click the **index.html** file or open it in your browser to view the login slider interface.
